# Minesweeper HAP

OpenHarmony ArkTS minesweeper application generated by OpenClaw.

## Application

- Bundle: `com.openclaw.minesweeper`
- Ability: `EntryAbility`
- Board: 10 x 10
- Mines: 15
- Tap to reveal a cell
- Long press to place or remove a flag

## Structure

- `AppScope/`: application configuration and icon
- `entry/src/main/ets/pages/Index.ets`: game state, mine placement and UI
- `entry/src/main/ets/entryability/EntryAbility.ets`: application entry
- `entry/src/main/resources/`: strings, colors, icon and page routing
- `build_minesweeper.sh`: board-side build, sign, install and launch script

Generated dependencies and build outputs are intentionally excluded. The
build script uses the repository-level `signature/` materials.
