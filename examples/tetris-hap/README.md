# Tetris HAP

OpenHarmony ArkTS Tetris application generated by OpenClaw.

## Application

- Bundle: `com.openclaw.tetris`
- Ability: `EntryAbility`
- Board: 10 x 20
- Seven tetromino types
- Move, rotate, soft drop and hard drop controls
- Line clearing, scoring and level-based speed
- Next-piece preview and game-over restart

## Structure

- `AppScope/`: application configuration and icon
- `entry/src/main/ets/pages/Index.ets`: game engine and ArkUI interface
- `entry/src/main/ets/entryability/EntryAbility.ets`: application entry
- `entry/src/main/resources/`: strings, colors, icon and page routing
- `build_tetris.sh`: board-side build, sign, install and launch script

Generated dependencies and build outputs are intentionally excluded. The
build script uses the repository-level `signature/` materials.
