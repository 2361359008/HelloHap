# Calculator HAP

OpenHarmony ArkTS calculator inspired by the supplied mobile calculator UI.

## Features

- Addition, subtraction, multiplication and division
- Decimal input and double-zero key
- Percentage conversion
- Backspace and all-clear
- Chained operations and repeated equals
- Division-by-zero error handling
- Responsive circular keypad

Bundle: `com.openclaw.calculator`

Use `build_calculator.sh` on the development board to build, sign, install,
and launch the HAP. The script reuses the repository-level `signature/`
materials.
