================================================================
自由发挥（自由开发）· 提示词草稿本
================================================================
用途：把你接下来要让 OpenClaw（还原助手）执行的提示词写在这里、随时往里攒，
方便复制粘贴到「自由开发」页底部输入框发给它执行。本文件不参与 HAP 编译。

部署后这份文件会被拷到板端日程工程根目录，可在 HAP 的「源码文件浏览器(Source)」里
直接打开查看（连同整个工程文件一起浏览）。

书写建议：每条提示词尽量写清「只改哪个文件、改哪一行/哪个组件、复用哪个已有状态、
改完做什么（编译/汇报）」，避免 OpenClaw 越界乱改其它文件。

----------------------------------------------------------------
【模板】单条还原 / 修改提示词
----------------------------------------------------------------
目标：<一句话说明要实现的效果>
只改动文件：/data/local/tmp/advanced-hapbuild/project/entry/src/main/ets/pages/Index.ets
定位：<要改的 Row / 组件特征，例如「类型」行 Text('类型') 那一行>
改法：<具体怎么改；复用哪个已有 @State；不新增状态、不改默认值、不碰别的行/文件>
完成后：执行 sh /data/local/tmp/advanced-hapbuild/build_advanced_with_env.sh 编译装板，
并回贴改动后的代码片段与所在行。

----------------------------------------------------------------
【示例①】还原「类型」下拉菜单（对应自由开发页 ① 按钮）
----------------------------------------------------------------
按主项目设计把「类型」行从静态展示还原成下拉菜单。
只改动 /data/local/tmp/advanced-hapbuild/project/entry/src/main/ets/pages/Index.ets：
- 在右侧值文本 Text(this.category) 上加 .bindMenu（不要加在整行 Row 或 Text('类型') 上）。
- 三个选项写回 this.category：普通 / 工作 / 学习。
- 传 { placement: Placement.Bottom }，让菜单从该值正下方弹出（不要弹到屏幕左上角）。
- 复用已有 this.category，不新增状态、不改默认值、不碰别的行。
改完执行 sh /data/local/tmp/advanced-hapbuild/build_advanced_with_env.sh 编译装板并回贴改动。

----------------------------------------------------------------
【示例②】还原「提醒」下拉与「响铃提醒」开关（对应自由开发页 ② 按钮）
----------------------------------------------------------------
在「更多设置」卡片中、紧邻「重复」行之后加回两行（先「提醒」下拉，再「响铃提醒」开关）：
- 「提醒」行：Text('提醒') + Blank() + Text(this.reminderMode) + Text('›')；.bindMenu 加在
  右侧值文本 Text(this.reminderMode) 上，选项写回 this.reminderMode：
  不提醒 / 准时 / 15 分钟前 / 1 小时前（半角空格一致），{ placement: Placement.Bottom }。
- 「响铃提醒」行：Text('响铃提醒') + Blank() + 开关（绑定 this.ringReminder）。
- 复用已有 this.reminderMode 与 this.ringReminder，不新增状态、不改默认值。
改完执行 sh /data/local/tmp/advanced-hapbuild/build_advanced_with_env.sh 编译装板并回贴改动。

【①】「更多设置」里新增一行「地点」
----------------------------------------------------------------
在「更多设置」分组里仿照 SettingRow('时区', 'GMT+08:00 北京', () => {}) 新增一行「地点」：
SettingRow('地点', this.location, ...)，并在顶部 @State 区加 @State location: string = ''。
复用既有 SettingRow 写法，不碰别的行/文件；改完编译装板并回贴改动。

----------------------------------------------------------------
【②】改设置行右侧取值文字颜色
----------------------------------------------------------------
把设置行右侧取值文字的浅灰色 '#777B82'（类型/重复/提醒的取值 Text）统一改成指定颜色；
标题文字色是 '#161616'。只改颜色值、不动结构与状态；改完编译装板并回贴改动。

----------------------------------------------------------------
【③】调整下拉菜单弹出方向
----------------------------------------------------------------
调整三处 bindMenu(..., { placement: Placement.Xxx })（类型/重复/提醒）里的
Placement.Bottom / Top / BottomRight，改变下拉菜单弹出方向。只改 placement 值、不动选项与回调；
改完编译装板并回贴改动。

----------------------------------------------------------------
【④】改编辑页顶栏文案/主题蓝
----------------------------------------------------------------
修改 EditorPage() 顶栏的 取消 / 新建 / 完成 三个 Text 的文案，或主题蓝 .fontColor('#1677FF')。
只改这三个 Text 的文案/颜色、不动按钮逻辑；改完编译装板并回贴改动。

----------------------------------------------------------------
【⑤】「提醒」下拉新增一项
----------------------------------------------------------------
在「提醒」的 bindMenu 数组里新增一项，例如
{ value: '3 天前', action: () => { this.reminderMode = '3 天前' } }。
复用既有 this.reminderMode，不新增状态、不改默认值；改完编译装板并回贴改动。

----------------------------------------------------------------
【⑥】改名称输入框 placeholder 与颜色
----------------------------------------------------------------
修改名称输入框 placeholder（'请输入日程名称' / '请输入要做的事情'）以及标题色
.fontColor('#171717')、占位色 .placeholderColor('#A4A7AC')。只改文案/颜色、不动输入逻辑；
改完编译装板并回贴改动。
